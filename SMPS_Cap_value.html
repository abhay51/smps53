Step 1: Here Step-down transformer use. suppose it secondary voltage 12 volts. Initially transformer is in RMS Value for capacitor value we need Pick to Pick voltage value .


Pick to Pick Voltage = 12 x 1.44

=16.968 Voltage


Step 2:Now, this voltage go into Diode where some Voltage drop. here in Positive half cycle 2 Diode work. and during negative half cycle other 2 Diode use. Suppose in Positive and negative half cycle voltage drop is 0.7 Volts.so, total voltage drop is 1.4 volts.

so, voltage out of bridge rectifier is 16.968-1.4 = 15.568 volts

this voltage is minimum voltage to capacitor charge. for that keep voltage level more like 30-40%., so, we take voltage value 20.23 volts

Step 3: we know , Q = C x V

where, Q = I x t

so, C = I x t / V (::Q = I x t)

Here, we use Voltage regulator IC 7805. for constant output voltage. its voltage range 7 to 25 Volts.

so, here min voltage is 7 volts and maximum 15.568 volts.
so, Voltage = 15.568-7 = 8.568 volts.


C = 1 x 10 mili second / 8.568 volts

C = 10000 / 8.568 Volts = 1167 Microfarad

Note : Here take 50% more range value capacitor.

so, C = 1750 Microfarad
